◈ ABOUT THIS PROJECT
TOB OS is a fully browser-based operating system built as a single HTML file — no downloads, no installs, no frameworks. It runs entirely in your browser using HTML, CSS, and vanilla JavaScript, with only fonts and Three.js loaded from CDN.

The project started as an experiment in how far a web UI could be pushed, and has since grown to include a real window manager, music player with folder access, video platform, mod system, app store with 15+ installable apps, themes, live wallpapers, user profiles, and more.

Every version is preserved here so you can see the evolution from the early builds all the way to the current Nexus Edition.
